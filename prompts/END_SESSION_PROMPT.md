# End Session Prompt

Before ending the session:
1. Summarize what changed.
2. Summarize evidence produced (paste it into docs/findings-log.md).
3. Update docs/decision-log.md if any decision was made; promote durable ones
   to long-term notes.
4. Update docs/current-stage.md with the latest state and bump last-verified.
5. Update docs/open-questions.md if new questions appeared.
6. Set status: active | parked | abandoned, with a one-line reason if not active.
7. List the next smallest useful step.
8. Commit changes or explain why not.
