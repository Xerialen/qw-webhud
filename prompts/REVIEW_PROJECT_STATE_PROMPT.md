# Review Project State Prompt

Review the repository documentation and answer:
1. What is the north-star goal?
2. What is the current active stage? Is its last-verified date stale?
3. What work has been done recently (per git, not just per docs)?
4. What decisions have been made?
5. What open questions remain?
6. What risks are increasing?
7. Are we drifting from the goal?
8. Should this project be parked or abandoned?
9. What should be the next smallest useful step?

Update docs if they are stale.
